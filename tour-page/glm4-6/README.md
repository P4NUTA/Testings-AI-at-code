# Tour Planner 55+ - GLM-4.6 Implementation

Комфортный планировщик туров для путешественников 55+ по Ленинградской области.

## Особенности

- **Полная доступность**: Маршруты с минимальными лестницами и пандусами
- **Максимальный комфорт**: Комфортный транспорт и места для отдыха
- **Четкий бюджет**: Прозрачный расчет всех расходов
- **Адаптация под возраст**: Специально для путешественников 55+
- **Поддержка погоды**: Альтернативные планы на случай дождя
- **Детерминированные результаты**: Воспроизводимые маршруты

## Технологический стек

- **Frontend**: React 18 + TypeScript + Vite
- **Стилизация**: Tailwind CSS
- **Интернационализация**: i18next (RU/EN)
- **Маршрутизация**: React Router Dom
- **Иконки**: Lucide React
- **Деплой**: Docker + nginx

## Функциональность

### Основные возможности

1. **Персонализированное планирование**: Учет возраста, мобильности, интересов и бюджета
2. **Оптимизация маршрутов**: Минимизация пересадок и времени в пути
3. **Оценка доступности**: Детальная информация о доступности каждого объекта
4. **Бюджетирование**: Расчет затрат с учетом всех категорий расходов
5. **Адаптация под погоду**: Альтернативные indoor-активности на случай дождя
6. **Экстренная информация**: Контакты служб помощи

### Алгоритм планирования

- Использует детерминированный генератор на основе seed
- Применяет хаверсин формулу для расчета расстояний
- Оптимизирует маршруты по критериям доступности и комфорта
- Учитывает предпочтения по транспорту и бюджету

### База данных аттракционов

18+ локаций с детальной информацией:
- Координаты и описание
- Критерии доступности (10 параметров)
- Цены и время посещения
- Погодная пригодность
- Уровень загруженности
- Дополнительные удобства
